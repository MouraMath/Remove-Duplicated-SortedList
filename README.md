# Remove Duplicates from Sorted List     || English - Portuguese

## English

This repository contains a Java solution for the LeetCode problem 83: Remove Duplicates from Sorted List.

### Problem Description

Given the head of a sorted linked list, delete all duplicates such that each element appears only once. Return the linked list sorted as well.

### Solution

The `RemoveDuplicatesFromSortedList` class provides a solution that:
- Traverses the linked list once
- Removes duplicate nodes in-place
- Handles edge cases like empty lists

### Usage

The `main` method demonstrates the solution with three examples:
1. A list with some duplicates
2. A list with duplicates at the beginning and end
3. An empty list (null)

### Time and Space Complexity

- Time Complexity: O(n), where n is the number of nodes in the list
- Space Complexity: O(1), as it uses constant extra space

## Português

Este repositório contém uma solução em Java para o problema 83 do LeetCode: Remover Duplicatas de uma Lista Ordenada.

### Descrição do Problema

Dada a cabeça de uma lista encadeada ordenada, delete todas as duplicatas de modo que cada elemento apareça apenas uma vez. Retorne a lista encadeada ordenada.

### Solução

A classe `RemoveDuplicatesFromSortedList` fornece uma solução que:
- Percorre a lista encadeada uma única vez
- Remove nós duplicados in-place
- Lida com casos extremos como listas vazias

### Uso

O método `main` demonstra a solução com três exemplos:
1. Uma lista com algumas duplicatas
2. Uma lista com duplicatas no início e no fim
3. Uma lista vazia (null)

### Complexidade de Tempo e Espaço

- Complexidade de Tempo: O(n), onde n é o número de nós na lista
- Complexidade de Espaço: O(1), pois usa espaço extra constante
